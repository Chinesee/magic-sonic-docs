# 代码格式

[环境变量](https://cli.vuejs.org/zh/guide/mode-and-env.html#模式) 可以让你灵活地切换状态。

## `VUE_APP_PAGE_TITLE`

## `VUE_APP_ROUTER_MODE`
## `VUE_APP_BASE_API`
API 接口域名
## `VUE_APP_REQUEST_BASE_URL`
接口统一标识
## `VUE_APP_PUBLIC_PATH`
如果要部署到类似「码云 Pages」或「Github Pages」，则此处更改为仓库名称
## `VUE_APP_OUTPUT_DIR`
打包路径
## `VUE_APP_ROUTE_PERMISSION_KEY_NAME`
路由权限的 key 字段名称
## `VUE_APP_PROJECT_URL`
项目仓库地址
## `VUE_APP_DOC_URL`
项目文档地址
